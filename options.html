  Tips and Instructions

Tip 1: Set locally stored personal image as background

You can select your own locally stored image as a background if you copy your image in the Images folder in the folder of the extension under the name "UserImage.*", where * is the format which can be .jpg, .jpeg, .png or .gif. If the image is not in one of these formats and you do not want to change its format, you can write in the URL text input "Images/image_name.format" after you have copied the image in the folder, "image_name" and "format" being placeholders for the actual name and format of the image. The path to the folder is:
a) i) For Windows XP: "C:\Documents and Settings\%USERNAME%\Local Settings\Application Data\Google\Chrome\User Data\Default\Extensions\<Extension ID>";
a) ii) For Windows 10/8/7/Vista: "C:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data\Default\Extensions\<Extension ID>";
b) For macOS: "~/Library/Application Support/Google/Chrome/Default/Extensions/<Extension ID>";
c) For Linux: "~/.config/google-chrome/Default/Extensions/<Extension ID>";
d) For Chrome OS: "/home/chronos/Extensions/<Extension ID>".
<Extension ID> is the name of the folder given by Chrome. You can find it in the Extensions menu, on the lower half of the extension's card, named ID. Extensions are found in the menu on the right side of the address bar, under the More Tools submenu or to the address chrome://extensions/. The ID is also found in this page's address: "chrome-extension://<Extension ID>/options.html".
Source (and a few other paths for other operating systems): stackoverflow.com/questions/...
Tip 2: Set an image from the internet as background

Tip 6: Uses of the extension action icon

If you click the extension action icon, that is, the icon with the extension's logo, which is placed to the right of the address bar, you can bookmark the current page of any given time to the New Tab page. Don't bookmark pages like "chrome://extensions" because the extension is not allowed to access pages that are on the local PC, but are not in the extension's folder.
